# Airbnb Price Prediction Using Linear Regression

This project predicts Airbnb listing prices using machine learning based on property features such as room type, bedrooms, bathrooms, reviews, and availability.

## Objective
The goal of this project is to estimate Airbnb prices using Linear Regression and understand how different features influence listing prices.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Workflow
- Data Cleaning
- Missing Value Handling
- Feature Selection
- Train-Test Split
- Feature Scaling
- Linear Regression Model Training
- Model Evaluation
- Visualization

## Model Used
- Linear Regression

## Evaluation Results
- MAE: 30.90
- RMSE: 59.58
- R² Score: 0.63

## Visualization
The scatter plot compares actual prices and predicted prices, showing that the model predicts most listings reasonably well.

## Files Included
- Airbnb_Price_Prediction.ipynb
- Report.pdf
- listings.csv

## Conclusion
The model explains 63% of the variation in Airbnb prices and gives good prediction performance for regular-priced listings.
